## Main Source:

```lua
--// Flame's Hub 🔥\\--
getgenv().Flames_Hub = "https://raw.githubusercontent.com/Lvl9999/Flames/main/Source";

pcall(function()
    loadstring(game:HttpGet(getgenv().Flames_Hub))();
end)
```
